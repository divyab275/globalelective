# globalelective

### Perform global elective allotment to S8 students 

## Getting Started
Fork the project and go to the directory where you cloned it.

### Prerequisites

You will need nodejs, mysql 8.0


### Installing

First create a database elective.

To start, change the config.json to suit your database credentials.

After than run the following command on your root

```
npm install
```

After the installation which will take some time, run the following command to start the server

```
npm start
```

Go to localhost:3000 where the site is running.
